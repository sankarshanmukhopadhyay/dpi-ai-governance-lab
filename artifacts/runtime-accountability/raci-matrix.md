# RACI matrix (starter)

R = Responsible, A = Accountable, C = Consulted, I = Informed

| Activity | System Owner | Model Operator | Data Steward | Assurer | Redress Authority |
|---|---|---|---|---|---|
| Tier classification | A | R | C | C | I |
| Data access approval | A | C | R | I | I |
| Model evaluation | A | R | C | C | I |
| Monitoring & drift | A | R | C | I | I |
| Incident response | A | R | C | I | I |
| Redress handling | A | C | I | I | R |
| Rollback / kill switch | A | R | C | I | I |
